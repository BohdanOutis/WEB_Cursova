# Інтернет-магазин автозапчастин

Web-додаток для продажу автозапчастин з можливістю перегляду каталогу, пошуку товарів та оформлення замовлень.

## Функціонал

- Перегляд каталогу автозапчастин
- Пошук товарів за назвою
- Фільтрація за категоріями та брендами
- Додавання товарів до кошика
- Оформлення замовлення
- Реєстрація та авторизація користувачів

## Технології

- HTML, CSS, JavaScript
- Node.js, Express.js
- Firebase (для авторизації та бази даних)

## Встановлення

1. Встановіть залежності:
```bash
npm install
```

2. Налаштуйте Firebase:
   - Створіть проект у Firebase Console
   - Отримайте конфігураційні дані
   - Замініть конфігурацію у файлі `public/js/auth.js`

3. Запустіть сервер:
```bash
npm start
```

4. Відкрийте браузер за адресою: `http://localhost:3000`

## Використання

1. Перегляньте каталог товарів на сторінці "Каталог"
2. Використовуйте фільтри для пошуку потрібних запчастин
3. Додайте товари до кошика
4. Оформіть замовлення, заповнивши форму
5. Зареєструйтеся або увійдіть для збереження історії замовлень
